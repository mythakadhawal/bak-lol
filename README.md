# Bak-Lol

Bak-Lol is a web application designed for university hostellers to connect, collaborate, and help each other. It's a platform where students can find peers in the same hostel, plan activities, share resources, and support each other in real life.

---

## Overview

Life in a hostel is full of opportunities to meet people, collaborate, and have fun—but sometimes finding like-minded peers or getting help can be difficult. HostelConnect aims to solve that by providing a safe, simple, and interactive platform for students within the university.

---

## Features

### Student Discovery
- Find students in your hostel or nearby hostels.
- Filter by department, year, and interests.
- View verified profiles.

### Activities
- Plan or join activities such as jamming sessions, parties, group study, late-night cravings, secret cooking sessions, hackathons, and discussions.
- Each activity has participant lists, dedicated chat, and status updates.

### Help & Requests
- Ask for items or help (books, notes, chargers, etc.).
- Offer help to other students.
- Direct messaging for contextual support.

### Public Requests
- Post open calls for study groups, project teammates, or event participation.

---

## Authentication & Privacy
- Login using university email or UG number (for first-year students without email yet).
- Collect essential profile info: year, department, hostel.
- Optional anonymous posting for sensitive requests.
- Full-profile visibility or limited display options for privacy.

---

## Tech Stack
- **Frontend:** React + TypeScript, HTML, CSS
- **Backend:** FastAPI (Python)
- **Database:** PostgreSQL (production) / SQLite (for testing)
- **Hosting:** Vercel (frontend), Render / Railway / Fly.io (backend)

---

## Design Principles
- Natural, soft, and approachable colors.
- Simple, distraction-free UI.
- Mobile-first design after web validation.
- Fast and action-oriented user experience.
